how it works

The generation loop.

1

Expand a persona

Start with a simple role and generate a richer profile with habits, projects, and context.

2

Build the computer

Create folders, file inventory, dependencies, artifacts, collaborators, and private references.

3

Run the simulation

Generate a long-horizon work history with daily activity, messages, revisions, and deliverables.

4

Read the retrospective

Summarize the run, score the outputs, and extract lessons from the generated environment.
