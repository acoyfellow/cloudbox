how it works

A compact version of the paper's loop.

1

Expand a persona

Start with a simple role and generate a richer profile with habits, projects, and context.

2

Build the computer

Create folders, file inventory, dependencies, artifacts, collaborators, and private references.

3

Run the simulation

Generate a long-horizon work history with daily activity, messages, revisions, and deliverables.

4

Read the retrospective

Summarize the run, score the outputs, and extract lessons from the generated work life.

use cases

Use Cloudbox when you need a complete work context.

Study long-horizon productivity

Create inspectable work histories with files, decisions, collaborators, and changing context.

Test agents in generated desktops

Give agents a realistic computer state and compare how they use evidence across files.

Create training environments

Practice with realistic productivity data without exposing private customer or company accounts.

Demo a product in context

Show your product inside a believable desktop with files, stakeholders, outputs, and evidence.
