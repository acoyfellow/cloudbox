how it works

A repeatable test run for real AI work.

1

Create a workspace

Generate a role, project brief, folder tree, source files, drafts, and collaborator context.

2

Give it to an agent

Ask the agent to complete a realistic assignment that depends on evidence across files.

3

Inspect the attempt

Review the work trail, generated files, revisions, messages, and final package.

4

Score the result

Turn the run into a scorecard with lessons you can reuse in future tests.
